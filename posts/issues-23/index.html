    两者的区别

    两者的区别的根源都在于 发布/订阅 模式中的 Event Channel(更多的叫法是 Broker 或 Event Bus) 模块.

    • 发布者/订阅者 双方不知道对方的存在, 存在一个第三方维护订阅关系, 称为 Broker / Event Bus; 观察者模式中的 Subject 记录了所有订阅者的信息.
    • 发布/订阅模式通常是异步的; 观察者模式通常是同步的.
    • 发布/订阅模式适用于跨应用; 观察这模式通常是同一个应用.
