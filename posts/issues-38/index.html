TCP "粘包" 备忘

    TCP 属于流式(stream)连接, 协议中并没有粘包的定义.

    基于 TCP 的应用发送端发送的数据通常有结构, 接收端需要从 TCP 数据 流 中切分出发送端不同次的数据以便解析数据.

    切分需要解决的问题即是粘包.

    粘包与 TCP 无关, 是应用层对数据切分问题的形象定义.

    半包 是 粘包 的其中一个场景.
