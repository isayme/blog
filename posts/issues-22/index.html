MongoDB 特性: Change Stream

    要求 MongoDB >= 3.6

    官方文档: Change Streams

    通过此特性可以做到对 collection/database/deployment 级别的数据变化的监听.

    应用场景

    官方文档: Use Cases 主要还是不太牵涉到业务的数据变更通知或同步类的场景.

    支持的事件有

    官方文档: Change Events

    • insert
    • delete
    • replace
    • update
    • invalidate
